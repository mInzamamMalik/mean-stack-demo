What is Encryption:

In cryptography, encryption is the process of encoding messages or information in such a way that only authorized parties can read it. Encryption does not of itself prevent interception, but denies the message content to the interceptor.


One-Way Encryption:

One-way encryption or cryptographic hash is cryptographic technique
in which "cryptographic hash functions" create some what called "Hash" as a result of our input data,output MAY (MUST in some systems) different for same input and these hash is considered "unable to decrypt", if any one have real data/password he can just check by comparing the hash and original data, this will return true if the password and hash matched and false if not matched,
NOBODY CAN ACCESS REAL DATA
these hash techniques are considered ideal for encryption passwords when passwords are stored in third party servers
